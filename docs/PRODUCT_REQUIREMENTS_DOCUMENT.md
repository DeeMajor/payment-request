# Utility Payment Request MVP

## Project Idea
A mobile-first payment request platform that allows individuals or small service providers to generate instant payment links or QR codes so customers can pay without sharing bank details.

Goal: Allow anyone to create a "Pay Me" request and get paid instantly.

---

## Target Users
Primary users include:

- Freelancers
- Small service providers
- Side-hustle sellers
- WhatsApp marketplace sellers

They need fast payments without repeatedly sharing banking details.

---

## Core Problem
Current payment process:

Customer asks to pay  
→ Seller sends banking details  
→ Customer performs EFT  
→ Customer sends proof of payment  
→ Seller manually checks bank  

Problems:

- Slow
- Manual
- Error-prone

Solution:

Generate a payment request → Customer scans QR or opens link → Payment confirmed automatically.

---

## First Feature (MVP)

Generate Payment Request.

User inputs:
- Amount
- Optional description

System generates:
- QR code
- Payment link
